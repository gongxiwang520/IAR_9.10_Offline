IAR Offline License Generator v1.4 by unis...
=============================================

The license generated with this keygen is not complete (except EWARM, EW430, EW8051),
Some files (such as the library sources) are placed in a password-protected RAR archive,
and the password stored inside license file in encrypted form.
Extract password possible only from the original license, bruteforce method too slow. 

So if anyone have original license and wish to help other peoples, 
please let me know (iarkgen@gmail.com),
I will explain you how to get it.

Actually, all what i need is a real license file, which stored here:
For example, The License file for AVR stored here:
%ProgramData%\IARSystems\LicenseManagement\LicensePackages\AVR\EW\1\Selected.package
The License file for STM8 stored here:
%ProgramData%\IARSystems\LicenseManagement\LicensePackages\STM8\EW\1\Selected.package
and so on...

=============================================

Лицензия генерируемая данным кейгеном не совсем полноценная (исключая EWARM, EW430, EW8051),
по причине того, что некоторые файлы (исходники библиотек) запакованы в отдельный 
самораспаковывающийся RAR архив защищенный 8-символьным цифро-буквенным паролем.
Пароль хранится в оригинальной лицензии в зашифрованном виде.
Так что, если кто-то обладает оригинальной лицензией и готов помочь другим, дайте знать, 
я объясню как извлечь пароль 
(как вы понимаете, он один для всех лицензий, так что по нему не определить источник).
iarkgen@gmail.com

Впрочем все что мне нужно это реальный файл лицензии,
это тектовый файл, который хранится тут: (например для AVR)
%ProgramData%\IARSystems\LicenseManagement\LicensePackages\AVR\EW\1\Selected.package
или тут для STM8:
%ProgramData%\IARSystems\LicenseManagement\LicensePackages\STM8\EW\1\Selected.package
и так далее...
